# 📚 Atividades - ADS (Java)

Este repositório contém atividades e exercícios desenvolvidos durante meus estudos no curso de **Análise e Desenvolvimento de Sistemas (ADS)**.

Os códigos foram criados com o objetivo de praticar **lógica de programação, resolução de problemas e fundamentos da linguagem Java**.

---

# Objetivo do projeto

O objetivo deste repositório é reunir exercícios realizados durante as disciplinas do curso, permitindo:

- Praticar lógica de programação
- Aplicar conceitos aprendidos em aula
- Melhorar a capacidade de resolver problemas com código
- Registrar minha evolução no aprendizado de Java

---

# Conteúdos praticados

Durante o desenvolvimento dessas atividades, foram trabalhados conceitos como:

- Estrutura sequencial
- Estruturas condicionais (`if` / `else`)
- Estruturas de repetição (`for`, `while`)
- Manipulação de variáveis
- Entrada e saída de dados
- Lógica de programação
- Resolução de exercícios propostos em aula
